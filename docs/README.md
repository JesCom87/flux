    Convergence - Flux

Convergence makes, Computers and Cellulars, that work in Tandem. They have an OperatingCore, inside a PrimeCard, inside a MainStructure, that work off ODP, called CC1. Replacing ISA, X86 and ARM. 

    Flux builds Flux

Flux is MachineOuterface (LowBack), to UserInterface (HighFront). It works with AbleWare (Firmware), and ProWare (Software). 

    Examples

• BIOS, Bootloader, MBR, Kernel, OS (System).
• Lexer, Parser, Assembler, Linker, Compiler (Core).
• Platform, Library, Framework, Stack (Modules).
